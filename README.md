# Workshop

In first screen, email and password are checked by filtering the entire user list for user login. 
In second screen, listing 52 user and have infinite scroll but only that list again and again. 
And in third screen, when clicked user in recyclerview, send data to profile screen. And you can edit profile only some field. If you click ok button the user infos save in sharedpreference.

## Using Features

 - Mvvm
 - LiveData
 - RxJava
 - Value Animator
 - Koin (Dependency Injection)
 - Kotlin Extensions
 - Retrofit
 - Picasso
 - Anko
 - Simple Unit & Instrumental Test
